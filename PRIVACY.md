# Cat Keyboard Lock Privacy

Cat Keyboard Lock processes keyboard and pointer events locally only while the
user has activated input locking. Event contents are not recorded, stored, or
sent to the developer.

The app uses macOS Accessibility permission to suppress selected input events.
Preferences such as lock duration, selected input types, trigger corner, trial
start, and debug overrides are stored locally in UserDefaults.

Purchases are processed by Apple and RevenueCat. Their services may process the
purchase and device/account metadata required to validate entitlement; Cat
Keyboard Lock does not receive payment-card details.

Support requests are initiated by the user through the links shown in the app.
